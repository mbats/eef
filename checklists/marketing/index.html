Marketing Checklist

MARKET-1

Check the branding of the update site, features and plugins

Setup

Add the update site to the Eclipse using the install new software

Instructions

Review the description of the update site and then click "more" below the description in order to review the metadata of the category. After that review the copyright, license, version number and general information of the features. Finally install the plugins in your Eclipse and check the branding of the feature in Eclipse along with the branding of the plugins

Expected Results

All the plugins and features have the proper version number, provider, license, copyright etc
