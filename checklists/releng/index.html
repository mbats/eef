Release Engineering Checklist

RELENG-1

Contribute the release to the aggregation build

Setup

Go to the simrel repository

Instructions

Ensure that EEF is properly contributed to the Simultaneous Release aggregation build

Expected Results

The Simultaneous Release aggregation build is successful

RELENG-2

Update the New & Noteworthy

Setup

Go to the New & Noteworthy page

Instructions

Update the page with the relevant information

Expected Results

The New & Noteworthy is up to date

RELENG-3

Update the Bugzilla

Setup

Go to the Bugzilla

Instructions

Close all the resolved issues, add the necessary versions in the administration interface and update the target milestones of the bugs that should be treated during the next release

Expected Results

The bugzilla is up to date

RELENG-4

Prepare the release review document

Setup

Go to the Eclipse project metadata

Instructions

Update the release review document with the relevant information, check that the issues related to the release are visible and send it to the PMC

Expected Results

The release review document is accepted by the Eclipse Modeling PMC

RELENG-5

Review the Software Repository Reports

Setup

Go to the Software Repository Report

Instructions

Check all the reports for any potential error regarding EEF

Expected Results

The report does not show any error involving EEF

RELENG-6

Review the IP Log

Setup

Go to the IP Log

Instructions

Check the report for any potential error regarding EEF

Expected Results

The report does not show any error involving EEF

RELENG-7

Tag the Git repository

Setup

Clone the Git repository

Instructions

Run git tag -a vX.Y.Z -m 'Version X.Y.Z' and git push origin --tags

Expected Results

The remote Git repository is tagged

RELENG-8

Create a dedicated branch on the repository

Setup

Clone the Git repository

Instructions

Run git branch vX.Y.Z and git push origin vX.Y.Z

Expected Results

The remote Git repository has a new branch for the maintenance of release

RELENG-9

Update the continuous integration

Setup

Go to the continuous integration server

Instructions

Create a job for the maintenance branch of the release

Expected Results

A new job is available

RELENG-10

Install in all supported versions of Eclipse

Setup

Download an Eclipse Modeling Package for all supported versions of Eclipse

Instructions

Install EEF using its release update site

Expected Results

EEF is properly installed in all supported versions of Eclipse

RELENG-11

Install from the zip

Setup

Launch a freshly downloaded Eclipse SDK

Instructions

Install EEF using the zip

Expected Results

EEF is properly installed

RELENG-12

Install from the Eclipse Marketplace

Setup

Launch a freshly downloaded Eclipse SDK

Instructions

Install EEF thanks to the Eclipse Marketplace

Expected Results

EEF is properly installed

RELENG-13

Install from the Eclipse Modeling UI

Setup

Launch a freshly downloaded Eclipse Modeling Package

Instructions

Install EEF thanks to the Eclipse Modeling UI

Expected Results

EEF is properly installed

RELENG-14

Install in Obeo Designer

Setup

Launch a freshly downloaded Obeo Designer

Instructions

Install EEF thanks to its stable update site

Expected Results

EEF is properly installed
